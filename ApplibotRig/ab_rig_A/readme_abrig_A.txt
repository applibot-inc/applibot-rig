Applibot Rig

--------------------------------------------------------------------------

・対応Mayaバージョン：Maya2023
※それ以前のバージョンでの使用に際して、動作不良が起こる可能性があります。

・リグの制御について
IKFKの切り替えやオプションパーツの可視性切り替え等のアトリビュートは、頭上の歯車状のコントローラーで制御します。
・注意事項

リグ内のコンストレイントノードに、アニメーションキーを打つと正常に動作しない可能性があります。pairblendノードの仕様により対策が難しい為、注意喚起としております。

--------------------------------------------------------------------------

-Compatible with Maya 2023
*When using the Applibot Rig with earlier versions of Maya, there is a possibility of malfunction.


-Rig Controls
Attributes such as switching IKFK and switching the visibility of optional parts are controlled by a gear-shaped controller on top of the head.


-Notes
It is possible that using an animation key on a constraint node in the rig may not function correctly. Unfortunately, it is difficult to prevent this problem due to the specifications of the pairblend node.
